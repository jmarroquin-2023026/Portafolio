Fennec

Desarrollado en JavaEE

Aplicación web desarrollada en Java EE para la gestión integral de compraventa de vehículos. Implementa un catálogo dinámico con búsquedas filtradas, autenticación segura, gestión de perfiles y sistema de transacciones. Arquitectura basada en JPA (Hibernate) y MySQL, destacando por su rendimiento optimizado y diseño responsive.

Tecnologías Utilizadas

Java EE
JPA
MySQL
