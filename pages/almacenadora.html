Almacenadora

Desarrollado con NodeJs y React

Aplicación web desarrollada en NodeJs y Ract, con esta seremos de llevar control de diversos productos gracias a su estructura inicial en backend, además de poder ver los productos que tenemos en inventario, así como agregar, editar y eliminar productos. También contamos con un sistema de autenticación para usuarios, lo que nos permite llevar un control de los productos que cada usuario tiene en su inventario.

Tecnologías Utilizadas

NodeJs
React
MongoDB
